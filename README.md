# Modelo formal de trazabilidad de medicamentos (VDM++)

Este repositorio contiene la especificación en VDM++ de un sistema de trazabilidad de medicamentos esenciales, usada en el artículo del curso de Métodos Formales.

## Contenido

- Clases VDM++ para `Medicamento`, `Lote`, `Evento` y `Sistema`.
- Clase de pruebas con escenarios de uso y verificación del modelo.

## Cómo ejecutar

1. Abrir el proyecto en Overture u otra herramienta compatible con VDM++.
2. Compilar todas las clases.
3. Ejecutar la clase de pruebas (por ejemplo, el método `testAll`) para verificar el comportamiento.
